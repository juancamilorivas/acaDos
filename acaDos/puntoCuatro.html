1. <body> <body/> Al contrario que la etiqueta de metadatos , todo lo que quieras mostrar en la página web debe ir recogido dentro de las etiquetas de apertura y cierre de . Este contenido será el que se muestre en la web.

  2. <nav> <nav/> Usadas para definir el contenido que será la sección de navegación de la web

  3. <main> <main/> Se usa para definir el contenido principal del documento. Solamente puede existir uno por documento.

  4. <section> <section/> Define una sección del documento

  5. <article> <article/> Define contenido independiente de la web.

  6. <aside> <aside/> Dentro de estas etiquetas suele alojarse el contenido adicional de la web. Suele ser contenido relacionado con la web pero de poca importancia

  7. <header> <header/> Se usan para definir la cabecera la página web. Suele contener el logotipo, menú de navegación, etc.

  8. <footer> <footer/> Usadas para definir el pie de página.

  9. <div> <div/> Etiqueta común utilizada para crear un contenedor genérico.

  10. <blockquote> <blockquote/> Se usan para indicar que el contenido es texto citado.
